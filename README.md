## Introduction
This project uses AI to turn topics, text blocks, or other input into complete quiz sets. Instead of spending hours crafting questions, you can generate quizzes instantly and export them as structured data.  
Its goal is to simplify learning content production for teachers, coaches, and digital course creators.

### What It Enables
- Converts prompts or topics into classroom-ready quiz items  
- Produces clean, structured quiz data you can plug into LMS platforms  
- Reduces manual writing time dramatically  
- Useful for training, onboarding, exam prep, and e-learning content generation

| Explanations Included | Each question includes an explanation to enhance learning. |
| Structured Output | Returns data in JSON for easy import into apps or LMS systems. |
| Flexible Input | Accepts various topic descriptions or content blocks. |

| Automated Workflow Ready | Can be chained with other Actors or automation tools. |

---
## What Data This Scraper Extracts
| Field Name | Field Description |
|------------|-------------------|
| question | The generated quiz question. |
| options | List of multiple-choice answer options. |
| answer | The correct answer chosen by the model. |
| explanation | Clarifies why the correct answer is correct. |
| topic | Topic or prompt used to generate the question. |
